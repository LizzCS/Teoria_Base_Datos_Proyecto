# Sistema de Presupuesto Mensual Personal

## Descripción

Este proyecto consiste en un sistema de presupuesto mensual personal que permite al usuario planificar, controlar y analizar sus finanzas personales de forma eficiente.

El sistema gestiona:

* Ingresos mensuales
* Gastos
* Obligaciones financieras
* Metas de ahorro

## Base de Datos

El sistema utiliza **Microsoft SQL Server** como gestor de base de datos.

La instalación se realizó desde el sitio oficial de Microsoft descargando la edición **Developer gratuita** de Microsoft SQL Server.

Posteriormente se instaló SQL Server Management Studio versión 22, que permite administrar la base de datos, ejecutar consultas y diseñar tablas.

## Objetivo del Proyecto

Desarrollar una herramienta que ayude a las personas a organizar sus finanzas personales, mejorar su control de gastos y fomentar el ahorro.

## Tecnologías utilizadas

* Microsoft SQL Server
* SQL Server Management Studio
* Python
